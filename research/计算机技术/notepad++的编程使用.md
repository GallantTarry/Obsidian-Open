少侠，Notepad++ 绝对是咱们程序员电脑里除了大型 IDE（比如你的 WebStorm、IDEA 或 Cursor）之外，最不可或缺的“轻量级瑞士军刀”。

它启动零秒延迟、极度轻量，非常适合用来临时看代码、改配置文件、处理庞大的报错日志，或者做复杂的文本批量替换。

作为顶层架构师，日常使用 Notepad++ 掌握以下这几个核心杀手锏功能就足够起飞了：

### 1. 列编辑模式（块选择）—— 最提效的封神功能

当你需要同时在几十行代码的最前面加上同一个单词，或者删除某一竖列的字符时，普通编辑器会让你抓狂，但在 Notepad++ 里只需一秒：

- **操作方法：** 按住键盘上的 **`Alt` 键** 不放，然后用**鼠标左键拖动**你需要选中的文本区域。
    
- **效果：** 你会看到光标变成了一条贯穿多行的竖线。此时你输入任何内容，这几十行会同时被修改。
    

### 2. 超级搜索与替换（支持正则表达式）

它的查找替换功能比 Windows 自带的记事本强大百倍。

- **快捷键：** `Ctrl + H` 唤出替换窗口。
    
- **高级用法：** 在左下角的“查找模式”中，一定要勾选 **“正则表达式” (Regular expression)**。
    
- **场景：** 比如你想把文本里所有的数字全部删掉，只需搜索框输入 `[0-9]`，替换框留空，点击全部替换，瞬间清爽。
    

### 3. 编码格式秒换（专治各种中文乱码）

Windows 环境下经常会遇到打开文本全是不认识的乱码符号，这通常是 ANSI 和 UTF-8 编码冲突导致的。

- **操作方法：** 点击顶部菜单栏的 **“编码” (Encoding)**。
    
- **日常排雷：** 如果打开是乱码，尝试点击“转为 UTF-8 编码”或“转为 ANSI 编码”，乱码通常瞬间原形毕露。
    

### 4. 必备插件生态 (Plugin Admin)

Notepad++ 是一个空壳，它的灵魂在于插件。

- **安装路径：** 点击顶部菜单栏 **“插件” (Plugins)** -> **“插件管理” (Plugin Admin)**。
    
- **强烈建议少侠安装的 3 个神级插件：**
    
    - **`Compare`：** 代码对比神器。左右分屏，自动用高亮颜色标出两份文件里哪怕一个标点符号的不同。
        
    - **`JSON Viewer`：** 处理接口数据必备。一键把挤成一团的乱码 JSON 字符串格式化成漂亮的树状结构。
        
    - **`XML Tools`：** 比如你之前处理的 `pom.xml`，如果排版乱了，用这个插件可以一键自动对齐。
        

### 5. 看大型日志不卡顿

很多时候服务器跑出来的 `.log` 报错日志动辄几百 MB 甚至上 GB，用普通记事本打开直接死机。

- **优势：** Notepad++ 的内存管理极其优秀，直接把几个 G 的日志文件拖进去，依然可以丝滑滚动、快速全局搜索报错关键词（如 `Exception` 或 `Error`）。
    

### 💡 附赠：日常最高频的几个快捷键

- **`Ctrl + D`**：向下复制并粘贴当前行（写重复配置时极为好用）。
    
- **`Ctrl + L`**：直接剪切当前行。
    
- **`Ctrl + Q`**：快速注释 / 取消注释当前行代码。
    
- **双击某个单词**：Notepad++ 会自动把你整篇文档里一模一样的单词全部高亮亮起，方便你追踪变量名。
    

少侠，你可以先在电脑上建一个 `.txt` 文本，随便敲几行字，按住 `Alt` 键拖动鼠标体验一下“列编辑”，保证你用一次就会爱上！
