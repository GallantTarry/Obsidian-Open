
    

### 方案二：降维打击，直接用 WSLg（极其推荐）

其实在开发自动化工具和写代码时，专门去维护一个完整的 Linux 桌面环境有时反而显得臃肿。Windows 11 最强大的地方在于它自带了 **WSLg** 技术——**它允许你直接把 Linux 的软件窗口开在 Windows 的桌面上！**

你不需要任何远程桌面连接。在那个黑色的 Ubuntu 窗口里，你可以直接安装并运行图形化软件。

试试在你的 Ubuntu 终端里输入：

Bash

```
sudo apt install thunar x11-apps
```

装完之后，直接在黑窗口里输入 `thunar`（这是一个 Linux 的文件管理器）或者 `xeyes`。

你会发现，Linux 的软件窗口会像原生的 Windows 软件一样直接跳出来，这种“无缝融合”的体验绝对会让你觉得惊艳。

### 方案三：使用开发者神器 MobaXterm

如果你希望有一个一站式的工具既能敲命令，又能传文件，还能看图形界面，强烈推荐你下载一个叫 **MobaXterm** 的免费软件（下载免费的 Home Edition 即可）。

它自带了 X Server。用它打开你的 Ubuntu 终端后，只要你在命令行里运行任何图形界面的指令，它会自动在 Windows 里帮你把画面渲染出来，不仅免去了配置远程桌面的麻烦，而且界面非常极客。

少侠，这层窗户纸换个角度照样能捅破。如果你只是想体验一下完整的桌面，用**方案一**；如果为了后续写代码、测工具更方便，**方案二**绝对是现在的最佳实践！
