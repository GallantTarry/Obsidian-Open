### 安装 Java 21 环境

在你的终端里，直接执行下面这两行命令（既然你是做开发的，我建议直接装 **JDK** 而不是仅仅装 JRE，这样以后不仅能跑 `.jar`，还能直接在 Linux 里排查和调试代码）：

Bash

```
sudo apt update
sudo apt install openjdk-21-jdk -y
```

装完之后，老规矩，敲一行命令验证一下：

Bash

```
java -version
```

只要终端里吐出了包含 `openjdk version "21...` 的字样，底层的运行环境就彻底就绪了。

接下来，你就可以像刚才说的那样，用 `cd` 命令进入你存放 `.jar` 文件的目录，然后执行：

Bash

```
java -jar 你的程序名字.jar
```

回车之后，你的可视化界面就会弹出来了。如果你的软件在 Ubuntu 桌面下出现任何 UI 组件排版错位，或者是中文字体显示成了小方块（Linux 默认可能缺少某些 Windows 下的常见中文字体），都可以随时拿报错和截图来找我，咱们现场解决。
