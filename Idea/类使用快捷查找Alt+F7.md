少侠，在 IntelliJ IDEA 中查看一个 Java 类在哪些地方被使用过非常简单。最核心的功能叫做 **Find Usages（查找用法）**。

你可以通过以下几种方式来查看，我按照推荐程度给你列了出来：

## 1. 快捷键全局查找（最推荐）

这是日常开发中最常用的方式，能够查得非常全：

- **Windows / Linux**: `Alt + F7`
    
- **Mac**: `Option + F7` (或 `⌥ + F7`)
    

**怎么用：** 将鼠标光标停留在你要查看的类名上，按下快捷键。IDEA 底部会弹出一个 **Find Usages** 窗口，它会按照模块、包、文件甚至具体的代码行，把所有引用了这个类的地方详细地树状列出来。

## 2. 悬浮窗快速查看

如果你只是想快速扫一眼都在哪里用过，不想在底部挤占屏幕空间打开新窗口，可以用这个“轻量级”的查找：

- **Windows / Linux**: `Ctrl + Alt + F7`
    
- **Mac**: `Cmd + Option + F7`
    

**怎么用：** 光标放在类名上按快捷键，代码上方会直接弹出一个小悬浮窗，列出所有的调用记录。你可以直接用键盘上下键选择，回车跳转，点击空白处悬浮窗就会消失。

## 3. 鼠标配合键盘（适合单手操作）

- **按住 Ctrl (Mac 是 Cmd) + 鼠标左键点击类名**。
    
- **效果：**
    
    - 如果这个类**只有一处**被使用，IDEA 会直接跳转到那一行代码。
        
    - 如果这个类有**多处**被使用，IDEA 会弹出一个类似下拉菜单的列表，让你选择要跳转到哪里。
        

## 4. 纯鼠标流

如果你暂时不想用键盘，可以直接在类名上点击 **鼠标右键**，然后在弹出的菜单里选择 **Find Usages**。

> **进阶小技巧：** 在底部的 Find Usages 窗口中，你可以点击左侧栏的漏斗图标（Filter），来过滤掉某些不需要看的引用（比如过滤掉注释里的引用、测试代码里的引用等），这样在阅读大型项目源码时效率会高很多。
