少侠，IntelliJ IDEA 中最常用的自动格式化（对齐）代码快捷键如下：

- **Windows / Linux:** `Ctrl` + `Alt` + `L`
    
- **macOS:** `Option` + `Command` + `L` (`⌥` + `⌘` + `L`)
    

按下这个快捷键，IDEA 就会根据你设置的代码规范，自动对齐缩进、调整空格和换行。

### 💡 实用进阶小贴士

在日常写 Java 或整理长篇 XML（比如你那份代理助手的 `pom.xml`）时，还可以配合以下几个快捷键使用，效率翻倍：

**1. 局部自动缩进：** `Ctrl` + `Alt` + `I`

如果你只想把当前光标所在的一行（或者选中的几行）对齐，不想大规模改变整个文件的格式，用这个最合适。

**2. 优化导入 (Optimize Imports)：** `Ctrl` + `Alt` + `O`

自动删掉代码里没用到的 `import` 语句，并按规范把需要的包重新排序。通常在按下格式化之后，顺手按一下这个，代码会非常干净。

**3. 唤出格式化高级菜单：** `Ctrl` + `Shift` + `Alt` + `L`

如果你不仅想对齐代码，还想顺便清理没用的引用、或者仅仅格式化你刚才修改过的那几行代码，按下这个组合键会弹出一个对话框让你精细勾选。

### ⚠️ 高频“避坑”预警

在 Windows 系统下，`Ctrl + Alt + L` 是一个**极度容易被其他软件占用的快捷键**。

如果你按了没反应，大概率是被**QQ**（锁定QQ快捷键）或者**网易云音乐**（喜欢/歌词快捷键）给拦截了。少侠遇到这种情况，去那俩软件的设置里把对应的全局快捷键删掉，IDEA 就能恢复正常了。



IntelliJ IDEA 的全局搜索快捷键取决于你的操作系统：

- **Windows / Linux:** `Ctrl` + `Shift` + `F`
    
- **macOS:** `Shift` + `Command` + `F`
    

**补充说明：**

- 这可以打开 **Find in Files**（在文件中查找）窗口，允许你搜索整个项目范围内的文本。
    
- 如果你想在全局范围内搜索类、文件或符号，可以使用 **Search Everywhere** 功能，快捷键是连续按两下 `Shift` 键。
